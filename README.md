# ReadAble

## Overview
ReadAble is a simple web tool designed to make reading easier and more accessible, especially for neurodivergent users (such as those with dyslexia or ADHD).

## Features
- Adjust font size and background color  
- Toggle a dyslexia-friendly font  
- Highlight key text  
- Text-to-speech (read aloud)  
- Basic AI-style text summary  

## How to Use
1. Paste text into the input box  
2. Adjust settings to fit your needs  
3. Use buttons to highlight, listen, or summarize  

## Technology
- HTML, CSS, JavaScript  
- Web Speech API  

## Future Improvements
- Real AI integration for better summaries  
- Keyword detection  
- Save user preferences
